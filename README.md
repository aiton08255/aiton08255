# Hey, I'm AIT

CS student building AI-powered creative tools. I like making things that feel like magic.

## What I Build

### [SceneForge Hybrid](https://github.com/aiton08255/sceneforge-hybrid)
**AI Text-to-3D Scene Generator** — Describe a scene in plain English, watch it build in real-time with Babylon.js. 130+ AI models, PBR materials, physics simulation, particle effects.

`TypeScript` `React` `Babylon.js` `NVIDIA NIM` `Zustand`

### [BrewBeats](https://github.com/aiton08255/brewbeats)
**AI Music Studio** — Text prompt to multi-track compositions. Full browser DAW with beat editor, AI advisor, ghost notes, 8 genre presets. MIDI + WAV export.

`TypeScript` `Next.js` `Tone.js` `Zustand` `Tailwind`

### [Spawner](https://github.com/aiton08255/spawner)
**AI Game Level Builder** — Describe a game world, AI generates a playable tilemap. 5 templates (Survivor, Dungeon, Arena, Adventure, Freeform). Edit, play-test, export as standalone HTML.

`TypeScript` `React` `Phaser 3` `Zustand` `Vite`

### [Self-Evo](https://github.com/aiton08255/self-evo) *(private)*
**Deep Research Engine** — 10-stage pipeline, 9 AI providers, 149 search sources, self-evolving. Learns from every query, absorbs skills, discovers new tools autonomously.

`TypeScript` `Hono` `SQLite` `React` `Tailwind`
